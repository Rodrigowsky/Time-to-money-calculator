Basic Extended Custom

The main goal of this project is to enable anyone to calculate how many hours do they need to work to reach a certain amount of money. The goal is to help people get a more clear sense of their finances and help them make financial decisions. This is an ongoing continuously improvement project so it might present data errors and inconsistencies. If you find any, please create an issue on github repository. This project is non commercial and is not affiliated with any company or organization. It is open-source and can be used by anyone.
